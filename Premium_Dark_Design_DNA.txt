‚óÜ 04 / MOTION & ANIMATION
The Motion System
Motion is where "nice site" becomes "premium site". Every element should feel like it arrives with intention. The
secret is one magic CSS curve.
The Magic Easing Curve
cubic-bezier(0.16, 1, 0.3, 1) ‚Üź "Expo Out" ‚ÄĒ use this everywhere
Things start fast then settle softly. This single value is responsible for 60% of the premium feel. Copy it into every
transition.
Base Scroll Animation (apply to every section element)
Initial state: opacity: 0; transform: translateY(40px);
Final state: opacity: 1; transform: translateY(0);
Transition: all 0.8s cubic-bezier(0.16, 1, 0.3, 1);

Flat dark sites look amateur. These techniques add depth and make the design feel crafted:
   Grain overlay ‚ÄĒ semi-transparent noise SVG over the whole page at 3‚Äď5% opacity. Makes it feel tactile, not

   Gradient vignette ‚ÄĒ radial gradient from transparent center ‚Üí black edges on hero sections.
   Glow effect ‚ÄĒ on accent elements: box-shadow: 0 0 40px ACCENT/30%
   Video blending ‚ÄĒ gradient overlay (transparent ‚Üí #080808) at bottom of video sections so they bleed into
